# 그래프


### 그래프 (Graph)

- 자료구조의 일종
- 정점(Node, Vertex)
- 간선(Edge): 정점 간의 관계를 나타낸다.
- G = (V, E)로 나타낸다.


### 경로 (path)

- ![경로](./image/경로.png)
- 정점 A에서 B로 가는 경
- A -> C -> D -> E -> B
- A -> B
- A -> C -> B
- A -> C -> E -> B


### 사이클 (cycle)

- 정점 A에서 다시 A로 돌아오는 경로 
- 경로 중에서 시작 = 도착인 것
- ![사이클](./image/사이클.png)
